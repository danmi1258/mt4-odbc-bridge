Class CGCursorEx is exception for cursor and cursor index classes.

Methods

long getCode();
Method returns code of exception

const char *getMsg();
Method returns textual description of exception.

Codes

CGCursorEx::eFileOpenCan't open specified file
CGCursorEx::eNoColumnsCursor doesn't contain columns
CGCursorEx::eUnknownTypeDatatype is unknown/unsupported
CGCursorEx::eWriteErrorWrite to file error
CGCursorEx::eInvalidRowRow index is invalid
CGCursorEx::eInvalidColumnColumn index is invalid
CGCursorEx::eIncompatibleTypeDatatype of column or of value is not compatible with operation/column
CGCursorEx::eReadErrorRead from file error
CGCursorEx::eBrokenOperation was broken
CGCursorEx::eInvalidColumnNameColumn name is invalid
